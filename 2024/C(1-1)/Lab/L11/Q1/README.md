# Basic Pointer

Then, please modify the code by introducing a pointer variable `ptr` to keep the address of `k` variable. Then, replace all `k` variable with a pointer variable `ptr`.

Sample input:
```
1
```

Sample output:
```
6
```

** Case 2 **

```
20
```
