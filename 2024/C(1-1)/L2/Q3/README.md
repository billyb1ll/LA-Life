# Comparing Three Numbers

Write a C program that determines whether three numbers are all the same, all different, or neither. 

**Note**: you can only compare two numbers at a time.

<hr>

**Case 1**

Sample input:
```
11 11 11
```

Sample output:
```
all the same
```

**Case 2**

Sample input:
```
1 3 5
```

Sample output:
```
all different
```

**Case 3**

Sample input:
```
1 5 5
```

Sample output:
```
neither
```
